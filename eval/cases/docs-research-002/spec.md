# Docs-Research Eval Case 002: React 19 Server Components

## Research Question
What is the production-ready picture of React Server Components (RSC) in
React 19, including data-fetching patterns, streaming, and limitations?

## Required Sources
- official docs of react.dev (RSC reference and 19.x changelog)
- at least 3 distinct domains (e.g. react.dev, vercel.com, github.com)
- exclude marketing/blog spam (no tutorial-aggregator sites, no LinkedIn posts)

## Required Outputs
- /docs/research/react-19-server-components.md (>=800 words)
- citations array with >=5 entries, each with author/title/url/access_date

## Acceptance
- AC-1: source_citation_count >= 5
- AC-2: word_count >= 800
- AC-3: each citation has author/title/url/access_date
- AC-4: explicitly covers RSC vs Client Components, async components, and
  streaming SSR boundaries introduced in React 19.

## Out of Scope
No code samples beyond illustrative snippets. No comparison with Solid/Qwik.
