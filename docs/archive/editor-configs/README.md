# editor-configs

Archived editor-specific configurations

---

## Purpose

This folder archives configuration files from various code editors that are no longer actively used in this project.

## Contents

- `.cursor/` - Cursor editor configuration
  - Archived on 2026-01-08 during cleanup
  - 116K, 10 files
  - Contains BMad agent rules and configurations
  - Not actively maintained

## Policy

**Preserved for reference only**

Editor configurations are archived when:
- The editor is no longer actively used
- Configuration is outdated or conflicting
- No recent activity detected

These configs may be useful for:
- Migrating to similar editors
- Understanding historical development setups
- Extracting useful configuration patterns
