## Summary

Fixed the legacy grant subagent leak in `hooks/posttool-allowlist-consume.py`.

When a subagent executed a Bash command authorized by a sentinel grant, the
legacy grant file (`/tmp/claude-bash-allowlist-<sid>.json`) was left on disk
because `consume_grant_for_posttool()` was guarded by `if not is_subagent_context(data):`.
The main agent could reuse this stale grant, bypassing single-use semantics.
