**Request ID**: 20260526-202532
**Created**: 2026-05-26T20:25:32Z

## Goal

Remove all dead code referencing the now-defunct `claude-bulk-allowed-*.flag` auth-flag concept from 3 hook files. The auth-flag mechanism was superseded by the sentinel-only flow: `/commit --bulk` has `disable-model-invocation: true`, so reaching the sentinel writer already proves human invocation. No separate auth flag is needed.

## Context

Task 20260526-052559 already fixed `scripts/write-bulk-commit-sentinel.py` to remove its auth-flag requirement. The surrounding hooks still reference the dead `claude-bulk-allowed-` pattern. This ticket removes that dead code and updates comments/docstrings to reflect the sentinel-only flow.

Correct flow: human invokes `/commit --bulk` -> orchestrator writes sentinel directly -> changelog-analyst commits -> guard checks sentinel -> ALLOW.
