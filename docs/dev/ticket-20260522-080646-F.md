## Goal

Rename 7 verification scripts in `hooks/tests/` from `_acN_verify.sh` / `_final_sweep.sh` convention to `test_` prefix convention, set executable permissions (755) on all renamed files, add `mktemp`+`trap` cleanup to the three files that write fixed `/tmp` paths, and fix one stale comment in `_final_sweep.sh`.

## Context

During the spec-20260518-225715 implementation cycle, dev created 7 verification scripts with underscore prefixes. The pre-existing naming convention in `hooks/tests/` is `test_*.sh` (established by `test_push_sentinel_abort.sh`, `test_allowlist_consolidation.py`, `test_commit_strip_dotfile_paths.py`, `test_cp_checkin.py`). All 7 files have `-rw-r--r--` (644) permissions — they are not executable. Three of them write to fixed `/tmp` paths without `mktemp`+`trap` cleanup, creating /tmp collision risk if invoked in parallel. One comment in `_final_sweep.sh` is stale.

## /tmp Cleanup Requirements

Three files write fixed `/tmp` paths and must be converted to mktemp+trap:

**`_ac1_verify.sh` (→ `test_ac1_verify.sh`)**: Uses `/tmp/push.sh.exec`. Replace with `mktemp` and clean up on EXIT.

**`_ac5_verify.sh` (→ `test_ac5_verify.sh`)**: Uses `/tmp/step7.md`. Replace with `mktemp` and clean up on EXIT.

**`_final_sweep.sh` (→ `test_final_sweep.sh`)**: Uses `/tmp/push.sh.exec`, `/tmp/push-analyst-phase7.py`, `/tmp/step7.md`. All three must be covered by a single trap block on EXIT. The Python snippet inside this file that reads `/tmp/push-analyst-phase7.py` hardcodes the path literal — the variable replacing it must be passed through to the inline Python string as well.
